## CPP module 05
A project focused on exception handling in C++, introducing try, catch and throw mechanisms to build robust error-resistant programs. The exercises involve creating custom exception hierarchies, propagating errors in a structured way and ensuring program stability in unexpected situations. Special attention is given to exception safety and proper resource management when errors occur. This module strengthens the understanding of error handling in C++ and best practices related to code reliability.
 
[Go to CPP 05](https://github.com/AlexisParder/42_Cursus/tree/main/milestone_05/cpp_5-9/CPP05)
 
## CPP module 06
A project dedicated to the different types of casts in C++ and type conversion. The exercises explore static_cast, dynamic_cast, reinterpret_cast and const_cast, highlighting their use cases, limitations and the risks associated with each. The module also introduces data serialization and runtime type detection via RTTI. This project enables mastery of type conversion in C++ while understanding the implications on program safety and behavior.
 
[Go to CPP 06](https://github.com/AlexisParder/42_Cursus/tree/main/milestone_05/cpp_5-9/CPP06)
 
## CPP module 07
An introductory project to generic programming in C++ through function and class templates. The exercises involve designing reusable, type-independent components, exploring template specialization and iteration over generic structures. The module emphasizes writing flexible and extensible code applicable to a wide variety of data types. This project builds a solid understanding of static metaprogramming and genericity in C++.
 
[Go to CPP 07](https://github.com/AlexisParder/42_Cursus/tree/main/milestone_05/cpp_5-9/CPP07)
 
## CPP module 08
A project focused on C++ Standard Library (STL) containers such as vector, list, map and stack. The exercises involve manipulating these data structures, implementing generic algorithms and using iterators to traverse collections. Special attention is given to choosing the right container for each context and the trade-offs between performance and readability. This module strengthens mastery of the STL and best practices for data structuring in modern C++.
 
[Go to CPP 08](https://github.com/AlexisParder/42_Cursus/tree/main/milestone_05/cpp_5-9/CPP08)
 
## CPP module 09
A project centered on STL algorithms and their application to real-world problems. The exercises explore sorting, searching and transformation algorithms applied to concrete data structures, notably through finance-inspired and advanced sorting challenges. The goal is to understand the inner workings of these algorithms, their complexities and their optimal conditions of use. This module consolidates mastery of STL algorithmic tools and their efficient application in C++.
 
[Go to CPP 09](https://github.com/AlexisParder/42_Cursus/tree/main/milestone_05/cpp_5-9/CPP09)
 
## Inception
A service virtualization and orchestration project based on Docker. The goal is to set up a multi-container infrastructure including an NGINX web server, WordPress with PHP-FPM and a MariaDB database, all configured manually without relying on pre-built Docker images. The project requires mastery of Dockerfiles, volumes, Docker networks and service configuration in an isolated environment. This project builds strong skills in modern system administration, containerized infrastructure deployment and web service hardening.
 
[Go to Inception](https://github.com/AlexisParder/42_Cursus/tree/main/milestone_05/inception)
 
## WebServ
A project consisting of implementing a fully functional HTTP/1.1 server in C++, capable of handling multiple simultaneous connections through non-blocking I/O and multiplexing mechanisms (select, poll or epoll). The goal is to reproduce the core behaviors of a web server such as NGINX: HTTP request parsing, handling of GET, POST and DELETE methods, configuration file support, CGI script execution and error handling. This project builds a deep understanding of network protocols, socket management in C++, thread-free concurrency and the low-level architecture of web servers.
 
[Go to WebServ](https://github.com/AlexisParder/42_Cursus/tree/main/milestone_05/webserv)
 
## IRC
A project consisting of implementing a fully functional IRC (Internet Relay Chat) server in C++, compatible with existing IRC clients. The goal is to handle multiple client connections, discussion channels, user and operator modes, as well as the core IRC protocol commands (JOIN, PART, PRIVMSG, KICK, INVITE, etc.). The server must operate with non-blocking I/O on a single thread. This project builds mastery of network communication protocols, socket management, text protocol parsing and the design of robust multi-client servers in C++.
 
[Go to IRC](https://github.com/AlexisParder/42_Cursus/tree/main/milestone_05/irc)
